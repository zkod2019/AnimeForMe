About AnimeForMe

AnimeForMe is website that I, Zaida Kodzoeva (M00736336), created for CW1 in the class Internet Scale Applications and Development. This class is taught at MDX for final year students, and the module code is CST3511. This website uses several APIs to get information on different anime, manga, and characters. All of this data can be viewed by users and added into their personal lists, to track what they are watching, what they are reading, their favorite characters, and the forums they've joined.
